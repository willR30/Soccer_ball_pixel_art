# Soccer Ball Pixel Art

This project is a small interactive web animation inspired by football. It features a pixel-style soccer ball that bounces around the screen and responds to the user in a playful way.

The experience is simple and visual, with a retro look and a lively atmosphere. It is designed to feel fun and a little nostalgic, almost like a tiny digital game scene.

## What you can do
- Watch the soccer ball move across the screen
- Interact with it using your mouse or touch
- Enjoy the colorful pixel-art style

## How to view it
Open the project in a web browser by loading the main HTML file.
